# Medicine Identifier Using Computer Vision

This project uses computer vision and OCR to identify medicine names from tablet strip images. The system processes images and extracts text using OCR technology.

## Technologies Used
Python
OpenCV
Tesseract OCR

## Features
Image preprocessing
Text extraction using OCR
Medicine name detection
Database matching

## Workflow
1. Capture medicine strip image
2. Preprocess image using OpenCV
3. Extract text using Tesseract OCR
4. Match detected text with medicine database

## Applications
Pharmacy automation
Medicine verification
Healthcare assistance systems
